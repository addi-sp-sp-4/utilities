Documentation


*DISCLAIMER*: I know putting everything in one big php file is not a great idea,
but this file is meant as a fast multifunctional script to deploy on web servers.

Parameter options:

Type parameter:

  1. structure
  2. dump
  3. console
  4. copy-db
  5. docs

The 'type' parameter is the parameter which decides the type
of the action you are executing.

structure:

    If the type parameter is 'structure', the structure of all the tables
    will be dumped

    Required parameters:
    1. host: Host of the server. If working with a local db, '127.0.0.1' as value should work
    2. user: User you are working with.
    3. pass: Password of the user you are working with.
    4. db: Database you are working with.
    Optional parameters:
    1. None

dump:

    If the type parameter is 'dump', the content of all the tables
    will be dumped

    Required parameters:
    1. host: Host of the server. If working with a local db, '127.0.0.1' as value should work
    2. user: User you are working with.
    3. pass: Password of the user you are working with.
    4. db: Database you are working with.
    Optional parameters:
    1. None

console:

    If the type parameter is 'console', you will get an SQL shell
    with optional callback.

    Required parameters:
    1. host: Host of the server. If working with a local db, '127.0.0.1' as value should work
    2. user: User you are working with.
    3. pass: Password of the user you are working with.
    4. db: Database you are working with.
    Optional parameters:
    1. query: Used to directly submit a query, without interface
    2. callback: to be used in conjunction with the query parameter.
      if set to true, you will recieve output

copy-db:

    If the type parameter is 'copy-db', a downloadable database backup is made.
    It is strongly advised to use this function with a tool like wget, otherwise your browser might crash

    Required parameters:
    1. host: Host of the server. If working with a local db, '127.0.0.1' as value should work
    2. user: User you are working with.
    3. pass: Password of the user you are working with.
    4. db: Database you are working with.
    Optional parameters:
    1. None

docs:

    If the type parameter is 'docs', you will get this page.

    Required parameters:
    1. None
    Optional parameters:
    1. None
